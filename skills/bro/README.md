# bro

Claude Code slash command that restates the agent's last message in plain human
language — no jargon, shorter, one human talking to another. Same content,
plainer words. Chat-only; does not edit files.

**Usage:** `/bro`

See [SKILL.md](./SKILL.md) for the rewrite rules.

## Install

```sh
npx skills add zcaceres/skills -s bro
```

Add `-g` for global install, or `-a <agent>` to target a specific agent.
