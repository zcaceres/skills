# cleanup-computer

Interactive file cleanup for Downloads, Desktop, and Documents. Goes file-by-file proposing delete, move, or keep. Use when user says 'clean up my computer', 'organize my downloads', 'help me declutter', 'what's in my Downloads', or 'file cleanup'.

## Layout

- `SKILL.md` — manifest + instructions (skills.sh standard)
- `scripts/` — executables the skill calls
- `references/` — docs the skill reads
- `assets/` — templates, samples

## Install

```
skills install @zcaceres/cleanup-computer
```

Or grab the tarball from the latest [GitHub release](https://github.com/zcaceres/skills/releases?q=cleanup-computer).
