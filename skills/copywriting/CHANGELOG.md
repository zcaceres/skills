# @zcaceres/skill-copywriting

## 1.0.0

### Major Changes

- Initial release: editor skill that refines rough text into clear, concise
  copy. Activates when the user asks to edit, tighten, or rewrite text.
  Ships core principles (cut the fat, active voice, varied rhythm, plain
  language, Anglo-Saxon words, verbs of movement, specificity, reframing,
  conversational tone), a Latinate→Anglo-Saxon swap table, a banned AI-tells
  list (delve, comprehensive, leverage, seamless, robust, streamline,
  empower, em-dashes for asides, etc.), and seven worked examples covering
  marketing copy, email cleanup, documentation, verbs of movement,
  specificity, reframing, and rhythm. Pure-prompt skill — no binaries.

  Ported from `~/.claude/skills/copywriting` with the body preserved
  verbatim.
