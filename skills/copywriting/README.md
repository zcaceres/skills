# copywriting

Editor skill that refines rough text into clear, concise copy. Activates when
the user shares text and asks to "edit this", "tighten this up", "make this
clearer", or "rewrite this".

See [SKILL.md](./SKILL.md) for the full instruction set: core principles,
Anglo-Saxon swap table, banned-words list, output format, and worked examples.

Pure-prompt skill — no scripts, no binaries.

## Install

```sh
npx skills add zcaceres/skills -s copywriting
```

Or grab the tarball from the latest
[GitHub release](https://github.com/zcaceres/skills/releases?q=copywriting).
