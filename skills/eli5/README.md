# eli5

Claude Code slash command that explains a concept like you're talking to a
curious five-year-old — short words, an everyday analogy, one idea at a time,
zero jargon. Explains the current conversation by default, or a topic passed
after the command. Pure-thinking tool — does not edit files.

**Usage:** `/eli5 [topic]`

See [SKILL.md](./SKILL.md) for input handling and the output format.

## Install

```sh
npx skills add zcaceres/skills -s eli5
```
