# perf-review

Analyze a full-stack web application for evidence-based performance bottlenecks

## Layout

- `SKILL.md` — manifest + instructions (skills.sh standard)
- `scripts/` — executables the skill calls
- `references/` — docs the skill reads
- `assets/` — templates, samples

## Install

```
npx skills add zcaceres/skills -s perf-review
```
