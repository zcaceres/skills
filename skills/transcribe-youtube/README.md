# transcribe-youtube

Download and transcribe YouTube videos to a markdown file using yt-dlp and Whisper

## Layout

- `SKILL.md` — manifest + instructions (skills.sh standard)
- `scripts/` — executables the skill calls
- `references/` — docs the skill reads
- `assets/` — templates, samples

## Install

```
npx skills add zcaceres/skills -s transcribe-youtube
```
